#  PortHunt — Dynamic Event Hub System

![PortHunt Banner](https://github.com/Sefard01/port-hunt/blob/main/Assets/images/logo.png?raw=true)

> **PortHunt** is a state-of-the-art interactive platform designed to bridge the gap between event organizers and attendees. Built with a hardcore tactical UI approach, it features clean geometry, interactive console grids, and an optimized user experience for tracking tech meetups, hackathons, and cultural events.

---

##  Built With

*   **HTML5** — Semantic system architecture.
*   **CSS3** — Custom variable matrix layout, responsive grid modules, and fluid animations.
*   **JavaScript (ES6+)** — Real-time tracking calculators, numerical tick engines, and viewport state monitors.
*   **FontAwesome Icons** — Vector asset visualization.

---

##  Core Features & Interaction Modules

*   **Interactive Terminal Grid:** Users can dynamically trigger matrix tab nodes to shift focus display pipelines instantly.
*   **Stage-Based Pipeline:** A micro-interaction tracking layout that clearly visualizes registration stages with smooth spatial feedback (`translateX`).
*   **Asymmetrical Tactical Gallery:** Custom masonry display area that locks image grayscale variations and slides up asset contexts using targeted crosshair cursors.
*   **Live Registration Progress Monitoring:** Real-time percentage tracking metrics that create organic high-retention engagement and FOMO triggers.
*   **Live Terminal Console Footer:** Embedded active pulse tracking nodes and an automated universal live system time clock (UTC).
